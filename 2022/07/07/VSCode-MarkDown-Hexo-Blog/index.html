MarkDown文件

VS Code 中 MarkDown 文件快捷键

快捷键 描述
ctrl + B 切换粗体
ctrl + I 切换斜体
alt + S 切换删除线
ctrl + shift + [\/] 切换标题级别
alt + shift + F 表格自动缩进

表格中内容对齐方式

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
